# Proyecto Final QA – Facebook Lite

## Integrante
Julieta Dastolfo

## Objetivo del testing
Validar las funcionalidades principales de la aplicación Facebook Lite,
asegurando su correcto funcionamiento en dispositivos Android con recursos
limitados y conectividad inestable.

## Alcance y entorno de prueba
Se realizaron pruebas funcionales, negativas y exploratorias sobre:
- Inicio de sesión
- Feed de noticias
- Publicaciones y reacciones
- Notificaciones
- Navegación general

Entorno:
- Android
- Testing manual
- Dispositivo real
- Conectividad WiFi y datos móviles

## Conclusión general
La aplicación es estable y cumple con su objetivo principal.
Los defectos encontrados no son críticos y no bloquean el uso,
por lo que el producto se considera apto para producción
con mejoras incrementales recomendadas.
